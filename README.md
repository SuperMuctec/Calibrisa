# Calibrisa

Stock Market, at a glance


## Installation

1. **Clone The Repositoty**

```shell
  git clone https://github.com/SuperMuctec/Calibrisa
```

2. **Make executable**

```shell
  chmod +x setup.sh
```

3. **Run the setup file**

```shell
  ./setup.sh
```

4. **To Stop**
```
  Ctrl + C in the terminal
```

5. **Rerun**
```
  ./run.sh
```
